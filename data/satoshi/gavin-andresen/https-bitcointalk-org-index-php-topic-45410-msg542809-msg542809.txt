What is the encryption method/algorithm used?  

dynamic iterated SHA512 hashing to derive a password key, AES256-CBC using the
password key to encrypt a master key, AES256-CBC using the master key to
encrypt the wallet keys.  

More details are in the doc/README file in the tree:  
https://github.com/bitcoin/bitcoin/blob/master/doc/README#L70
